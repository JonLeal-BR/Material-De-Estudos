INFORMÁTICA EXTREME — ULTRA DENSA

1. HARDWARE

Definição: parte física do computador.

Componentes principais:

  • CPU (processamento)
  • Memória RAM (temporária)
  • HD/SSD (armazenamento permanente)
  • Placa-mãe (integração)

RAM: volátil → perde dados ao desligar.

HD/SSD: não volátil → mantém dados.

CESPE gosta: inverter conceitos de memória.

2. SOFTWARE

Definição: conjunto de programas.

Classificação:

  • Sistema (Windows, Linux)
  • Aplicativo (Word, Excel)
  • Utilitário (antivírus)

Sistema Operacional:

  • Gerencia hardware
  • Gerencia processos
  • Gerencia arquivos

Ponto CESPE: confundir software com hardware.

4. INTERNET E REDES

Internet: rede mundial.

Protocolos:

  • HTTP / HTTPS
  • FTP
  • TCP/IP

Diferença crítica:

  • HTTP → sem criptografia
  • HTTPS → com criptografia

IP: identifica dispositivo na rede.

DNS: traduz nome em IP.

CESPE gosta: trocar função de DNS e IP.

5. SEGURANÇA DA INFORMAÇÃO

Princípios:

  • Confidencialidade
  • Integridade
  • Disponibilidade

Ameaças:

  • Vírus
  • Worm
  • Trojan
  • Phishing

Diferenças importantes:

  • Vírus → precisa de ação
  • Worm → se espalha sozinho
  • Trojan → disfarçado

CESPE cobra: trocar definições.

7. ARMAZENAMENTO E NUVEM

Cloud computing: armazenamento remoto.

Vantagens:

  • Acesso remoto
  • Backup
  • Escalabilidade

Risco: dependência de internet.

8. CONCEITOS AVANÇADOS (NÍVEL PROVA)

Backup: cópia de segurança.

Tipos:

  • Completo
  • Incremental
  • Diferencial

Diferença chave:

  • Incremental → menor tempo / restauração mais complexa
  • Diferencial → cresce ao longo do tempo

Firewall: controla tráfego de rede.

Criptografia: protege dados.

CESPE ama: inverter conceitos de backup.
